# 光合 · 美好记录

**用一句话，记录今天的美好。**  
每一条记录，都是一缕照进土地的阳光——让你的森林，一棵一棵地生长。

---

## 产品背景

这个项目的灵感来自一条朋友圈：朋友在本子上用一行字记录美好小事，两个月写了 40 条，说要写够 100 条。那条朋友圈收获了大量的点赞和「好温暖」的评论。

**洞察**：人们渴望记录生活中的小美好，但门槛不能高——一句话就够，不需要写日记那么重。缺少的只是一个足够轻盈、又有成就感反馈的产品。

光合的答案是：每记录一件小事 = 一缕阳光 = 长出一棵小树。30 棵小树可以聚合成一棵大树。美好在积累，森林在生长。

---

## 功能特性

**记录页**
- 一句话输入，快速记录美好瞬间
- 支持语音输入（调用 Web Speech API，中文识别）
- 记录列表按时间倒序排列，带日期和时间戳
- 一键导出手帐 PDF（横线纸张风格，支持多页打印）

**森林页**
- 等距视角（Isometric）立体世界，柔和 Minecraft 画风
- 小树和大树以 Canvas 2D 绘制，生长在立体草地方块上
- 手指拖拽可自由移动视角，世界随记录增多自动扩展
- 累积 30 棵小树后，可一键聚合成一棵大树

---

## 技术实现

| 项目 | 说明 |
|------|------|
| 技术栈 | 原生 HTML / CSS / JavaScript，零依赖，零构建 |
| 数据存储 | localStorage（`gh_records` / `gh_bigtrees`） |
| 森林渲染 | Canvas 2D + Painter's Algorithm 等距投影 |
| 语音输入 | Web Speech API（iOS Safari / Android Chrome 原生支持） |
| PDF 导出 | `window.open()` 新窗口 + `window.print()` |
| PWA | 支持添加到手机主屏幕，可离线使用 |
